BoshStemcells.com

You can use the following URL structure to get the latest BOSH stemcell for your IaaS:
https://boshstemcells.com/[iaas]

If you need a specific version or stemcell line you can just append that to the end of the URL:
https://boshstemcells.com/[iaas]/[versionOrStemcellLine]

If you need a specific stemcell line with a specific version you can also append that to the end of the URL:
https://boshstemcells.com/[iaas]/[stemcellLine]/[version]
